# 功能介绍及展示

功能介绍及展示为开发版页面，与稳定版仓库有一定差异。

## 💬 AI 对话

支持 OpenAI Chat 格式，后台可自定义模型名称、头像、介绍、代理、key、积分扣除方式、文件上传模式等参数。
